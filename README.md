# TcpClientServerApp

Async TCP Client / Server App (Protocol based) using C++ Boost::Asio

## How to build?

Prerequisites: CMake

Go to the folder that you want to build ([`Server`](Server)/[`Client`](Client)).

```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## How to run?

Make sure you're `build` folder i.e. `Server/build` and `Client/build`.

### Server

```
$ ./Server
```

### Client

```
$ ./Client
```
